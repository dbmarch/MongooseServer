# MongooseServer


### Dependencies
This project includes a few 3rd party open source packages:
- jsoncpp
- mongoose


### BUILDING

```
    git clone https://github.com/dbmarch/MongooseServer.git

    cd MongooseServer

    git submodule update --init

    make
```

### RUN SERVER

To start the server:

```
  bin/MongooseServer
```

    Browse localhost:8000
